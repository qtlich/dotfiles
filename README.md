# dotfiles
Some configurations for my workflow.
+ **OS**: Arch Linux
+ **Home directory**
  - build
  - devel
  - download
  - audio
  - pic
  - vid
+ **Window system**: [Xorg](https://www.x.org/wiki/)
+ **Window manager**: [dwm](https://dwm.suckless.org/)
+ **Wallpaper manager**: [feh](https://feh.finalrewind.org/)
+ **Fonts**
  - [Terminus](https://terminus-font.sourceforge.net/)
  - [Jetbrains Mono Nerd Font](https://www.nerdfonts.com/font-downloads)
  - [Material Design Iconic Font](https://zavoloklom.github.io/material-design-iconic-font/)
+ **Status bar**: [slstatus](https://tools.suckless.org/slstatus/)
+ **App launcher**: [dmenu](https://tools.suckless.org/dmenu/)
+ **Lockscreen**: [slock](https://tools.suckless.org/slock/)
+ **Terminal emulator**: [alacritty](https://alacritty.org/)
+ **Terminal multiplexer**: [tmux](https://github.com/tmux/tmux/wiki)
+ **Text editor**: [nvim](https://neovim.io/) (NeoVim)
+ **Web browser**: [librewolf](https://librewolf.net/)
+ And other utilities
